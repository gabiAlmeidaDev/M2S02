PROJETO LABPCP - (Sistema de Gestão Educacional)

Este sistema é uma API RESTful destinada a gerenciar operações em uma instituição educacional, incluindo o gerenciamento de alunos, professores, cursos, turmas e notas. O sistema visa simplificar a gestão de instituições educacionais permitindo o controle centralizado de dados e operações relacionadas. Facilita o acesso e a gestão de informações, melhora a comunicação entre professores e alunos, e permite uma análise mais eficaz do progresso educacional.

Tecnologias e Técnicas Utilizadas: Java/Spring Boot; Spring Security & JWT; PostgreSQL; Maven; Logback para Logging; Docker
